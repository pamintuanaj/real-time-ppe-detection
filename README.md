# real-time-ppe-detection
## Project Overview
Workplace safety is a major concern in industries such as construction, manufacturing, and logistics where workers are exposed to hazardous environments. Personal protective equipment (PPE) such as hard hats, safety vests, and masks helps reduce the risk of injuries and accidents. However, monitoring whether workers consistently wear PPE can be difficult when done manually, especially in large workplaces with many employees.

This project proposes a computer vision–based system that automatically detects PPE in workplace images using deep learning object detection models. By using convolutional neural networks (CNNs), the system will detect PPE items such as helmets, masks, and safety vests and highlight them using bounding boxes. The goal is to assist safety officers in improving workplace monitoring and ensuring better compliance with safety regulations.
